# Student Record Management System (ICT 143-2)

This is a comprehensive Java desktop application developed as a practical assignment for the Data Structures and Algorithms module at Uva Wellassa University. The system provides an efficient way to manage student records, handle admissions, and rank students based on their GPA using custom-implemented data structures.

## 🚀 Key Features & Data Structures Used

To achieve optimal performance and demonstrate algorithmic concepts, the following custom data structures have been implemented without using built-in Java collections:

*   **Student Records Management (Hash Table):** Student data is stored and retrieved efficiently using a **Custom Hash Table** for $O(1)$ fast search by Student ID.
*   **Admission Processing (Queue):** A **Custom Queue** manages the incoming student admission requests on a First-In, First-Out (FIFO) basis.
*   **Undo Functionality (Stack):** A **Custom Stack** tracks user actions (Add/Delete), allowing users to undo their last action seamlessly.
*   **GPA Ranking & Search (Binary Search Tree / Sorting):** A **Custom BST** and efficient sorting algorithms are used to rank students by their GPA and display them in the system.
